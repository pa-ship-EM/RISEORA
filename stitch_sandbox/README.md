# RiseOra - Stitch Sandbox Prototype

This directory contains a standalone prototype of the RiseOra landing page, designed using Stitch with Google.

## Design Specs
- **Background**: Navy (`#0A192F`)
- **Accents**: Gold (`#D4AF37`)
- **Typography**: Merriweather (Serif) & Inter (Sans-Serif)

## Directory Structure
- `index.html`: Main landing page structure.
- `styles.css`: Custom CSS for the Navy/Gold theme and modern animations.
- `assets/`: Generated design assets (placeholders for now).

## How to View
Open `index.html` in any modern web browser.
