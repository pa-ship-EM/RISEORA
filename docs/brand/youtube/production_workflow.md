# AI Content Production Workflow

A step-by-step guide to producing high-end YouTube content using the "RiseOra Method."

## Phase 1: Scripting (The Intelligence)
1. **Source Feeding**: Upload `notebooklm_source.md` to NotebookLM.
2. **Drafting**: Use the "Chat" feature to ask: "Write a 5-minute YouTube script for RiseOra on the topic of [X] using the brand tone."
3. **Human Edit**: Take the AI output and add your personal touch, "Big Brother" phrasing, and specific story beats.

## Phase 2: Visuals (The Aesthetic)
1. **B-Roll Generation**: Use Midjourney or Dall-E for stills, then animate them with Luma Dream Machine or Runway Gen-2.
2. **Key Visuals**: For RiseOra, focus on abstract "elevation" imagery. For Personal, use "desk setup" or "nature" clips.
3. **Face-Swap/Avatars (Optional)**: If you want an AI host for the RiseOra channel, use HeyGen or Synthesia.

## Phase 3: Audio (The Voice)
1. **Voice Cloning**: Clone your own voice (for the personal channel) or use a high-end AI voice (ElevenLabs) for RiseOra.
2. **Music**: Use AI music generators (Suno/Udio) to create custom "RiseOra Ambient" tracks.

## Phase 4: Assembly (The Craft)
1. **Editing**: Assemble in Premiere Pro or CapCut.
2. **Color Grade**: Apply a "Sage/Bronze/Minimal" LUT to ensure brand consistency.
3. **Review**: Ensure the pacing is "Elegant and Mature" (Slow-motion clips, punchy text).

---

## Tools Recommendation List
- **Scripting**: NotebookLM, Claude (for coding prompts), Antigravity (for project management).
- **Images**: Midjourney (v6.1), Leonardo.ai.
- **Video Motion**: Runway Gen-3, Luma, Pika Labs.
- **Audio**: ElevenLabs, Udio.
- **Editing**: CapCut (Desktop version is fast and has great AI features).
